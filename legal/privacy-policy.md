## BAGIAN I: KEBIJAKAN UMUM (WEBSITE & APLIKASI)

### 1. FILE LOG

Website HKBP Perawang mengikuti prosedur standar dalam menggunakan file log. File-file ini mencatat pengunjung ketika mereka mengunjungi situs web. Semua perusahaan hosting melakukan ini dan merupakan bagian dari analitik layanan hosting.

**Informasi yang dikumpulkan oleh file log termasuk**:
- Alamat protokol internet (IP)
- Jenis browser
- Penyedia Layanan Internet (ISP)
- Cap tanggal dan waktu
- Halaman rujukan/keluar
- Jumlah klik

Informasi ini **tidak terkait dengan informasi apa pun yang dapat diidentifikasi secara pribadi**. Tujuan dari informasi tersebut adalah untuk menganalisis tren, mengelola situs, melacak pergerakan pengguna di situs web, dan mengumpulkan informasi demografis.

---

### 2. COOKIES DAN WEB BEACON

Seperti website lainnya, Website HKBP Perawang menggunakan **'cookies'**. Cookies ini digunakan untuk menyimpan informasi termasuk preferensi pengunjung, dan halaman-halaman di situs web yang diakses atau dikunjungi pengunjung.

**Tujuan Cookies**:
- Mengoptimalkan pengalaman pengguna dengan menyesuaikan konten halaman web kami
- Menyimpan preferensi browser dan informasi lainnya

Untuk informasi yang lebih umum tentang cookie, silakan baca artikel ["What Are Cookies"](https://www.cookieconsent.com/what-are-cookies/) di situs web Cookie Consent.

**Untuk detail lengkap tentang cookies**, lihat **[Kebijakan Cookie](https://aplikasi.hkbpperawang.org/legal/cookie-policy)** kami.

---

### 3. PRIVASI IKLAN PIHAK KETIGA

Anda dapat berkonsultasi dengan daftar ini untuk menemukan Kebijakan Privasi masing-masing mitra HKBP Perawang.

Server iklan pihak ketiga atau jaringan iklan menggunakan teknologi seperti cookie, JavaScript, atau Web Beacon yang digunakan di masing-masing iklan dan tautan yang muncul di HKBP Perawang, yang dikirim langsung ke browser pengguna. Mereka secara otomatis menerima alamat IP Anda saat ini terjadi.

**Teknologi ini digunakan untuk**:
- Mengukur keefektifan kampanye iklan mereka
- Mempersonalisasi konten iklan yang Anda lihat di situs web yang Anda kunjungi

**Perhatikan bahwa** Website HKBP Perawang **tidak memiliki akses atau kontrol** atas cookie yang digunakan oleh pengiklan pihak ketiga ini.

---

### 4. KEBIJAKAN PRIVASI PIHAK KETIGA

Kebijakan Privasi Website HKBP Perawang **tidak berlaku** untuk pengiklan atau situs web lain. Karenanya, kami menyarankan Anda untuk berkonsultasi dengan masing-masing Kebijakan Privasi dari server iklan pihak ketiga ini untuk informasi yang lebih rinci.

**Anda dapat memilih untuk menonaktifkan cookie** melalui opsi browser individual Anda. Untuk mengetahui informasi lebih rinci tentang manajemen cookie dengan browser web tertentu, dapat ditemukan di situs web masing-masing browser.

**Layanan Pihak Ketiga yang Kami Gunakan**:
- Google Analytics
- Firebase (Google)
- Microsoft Clarity
- RankMath SEO

Untuk informasi lengkap, lihat **Kebijakan Cookie** kami.

### 5. KOMENTAR (WEBSITE)

Ketika Anda meninggalkan komentar di situs, kami mengumpulkan:
- Data yang ditampilkan di formulir komentar
- Alamat IP Anda
- String agen pengguna browser untuk membantu deteksi spam

String anonim yang dibuat dari alamat email Anda (juga disebut hash) dapat diberikan ke layanan **Gravatar** untuk melihat apakah Anda menggunakannya. Kebijakan privasi layanan Gravatar tersedia di [sini](https://automattic.com/privacy/).

**Setelah persetujuan atas komentar Anda**, gambar profil Anda dapat dilihat oleh publik dalam konteks komentar Anda.

Kerahasiaan data dan Data Pribadi Anda menjadi perhatian utama bagi Kami. Kami melakukan usaha terbaik dan prosedur untuk melindungi dan menjamin Data Pribadi.

**Namun**, kami **tidak dapat menjamin 100%** bahwa sistem Kami tidak dapat ditembus oleh virus, malware, dan gangguan dari pihak ketiga.

**Saran Keamanan**:

Untuk detail lengkap tentang langkah-langkah keamanan kami, lihat **[Kebijakan Keamanan](https://aplikasi.hkbpperawang.org/legal/security-policy)**.

## BAGIAN II: APLIKASI MOBILE

### 9. INFORMASI YANG KAMI KUMPULKAN (APLIKASI)

Aplikasi mobile HKBP Perawang dapat mengumpulkan beberapa jenis informasi:

#### 9.1. Informasi Pribadi (Khusus Admin/Editor)
Informasi yang dapat diidentifikasi secara pribadi, seperti:
- Nama lengkap
- Alamat email
- Nomor telepon
- Foto profil (jika Anda memilih untuk mengunggahnya)
- Informasi lain yang Anda berikan secara sukarela

**Catatan**: Pengguna umum **TIDAK** diminta untuk mendaftar atau memberikan informasi pribadi.

#### 9.2. Informasi Non-Pribadi
- Jenis perangkat mobile
- Sistem operasi perangkat
- Pengidentifikasi perangkat unik
- Alamat IP
- Data penggunaan aplikasi
- Informasi diagnostik

#### 9.3. Data Lokasi (Opsional)
Dengan persetujuan Anda, kami dapat mengumpulkan informasi tentang lokasi Anda untuk:
- Menampilkan lokasi gereja
- Memberikan petunjuk arah ke lokasi kegiatan gereja
- Menampilkan acara gereja di sekitar Anda

#### 9.4. Informasi Log
- Halaman atau fitur aplikasi yang Anda kunjungi
- Waktu dan tanggal akses
- Waktu yang dihabiskan di halaman tertentu
- Informasi perangkat dan koneksi

### 13. PENYIMPANAN DAN KEAMANAN DATA

#### 13.1. Lokasi Penyimpanan
Data pengguna disimpan di server **Firebase** yang berlokasi di:
- **us-central1** (Iowa, Amerika Serikat)
- **asia-southeast1** (Singapura)

#### 13.2. Durasi Penyimpanan
Untuk detail lengkap tentang berapa lama kami menyimpan data, lihat **[Kebijakan Retensi Data](https://aplikasi.hkbpperawang.org/legal/data-retention-policy)**.

**Ringkasan**:
- Cache aplikasi: 6 jam - 14 hari (varies)
- Data akun Admin/Editor: Selama aktif + 4 bulan setelah tidak aktif
- Log error: 30 hari
- Analytics: 14 bulan

#### 13.3. Langkah Keamanan
- Enkripsi TLS 1.3 (in-transit)
- Enkripsi AES-256 (at-rest)
- Firebase Security Rules
- Monitoring dan audit logs

### 14. PENGGUNAAN DATA PENGGUNA GOOGLE (GOOGLE USER DATA)

**Kebijakan ini hanya berlaku untuk Aplikasi HKBP Perawang.**

Aplikasi kami **tidak memiliki fitur Login dan Penggunaan Data User secara Publik**, karena fitur Login **hanya kami berikan untuk Admin** yang menangani aplikasi.

#### A. Data yang Kami Akses
Jika Anda (Admin/Editor) memilih untuk menggunakan fitur Login dengan Google, aplikasi kami dapat mengakses:
- **Informasi Profil Dasar**: Nama lengkap, alamat email, dan foto profil

#### B. Cara Kami Menggunakan Data
- **Otentikasi**: Memverifikasi identitas Anda dan memungkinkan login tanpa kata sandi baru
- **Personalisasi**: Menampilkan nama dan foto profil Anda di dashboard
- **Komunikasi**: Mengirimkan notifikasi terkait akun ke alamat email Anda

**Kami TIDAK**:
- ❌ Menggunakan data pengguna Google untuk tujuan periklanan
- ❌ Menjual data kepada pihak ketiga

#### C. Penyimpanan dan Pembagian Data
- Data yang kami terima dari Google disimpan secara aman sesuai standar keamanan industri
- Kami **tidak membagikan** data pengguna Google dengan pihak ketiga, kecuali jika diwajibkan oleh hukum

#### D. Kepatuhan Kebijakan Penggunaan Terbatas (Limited Use Policy)
Penggunaan dan transfer informasi yang diterima dari Google API ke aplikasi lain akan mematuhi [Kebijakan Data Pengguna Layanan Google API](https://developers.google.com/terms/api-services-user-data-policy), termasuk persyaratan Penggunaan Terbatas.
